Project Overview

This project focuses on improving catheter manipulation in endovascular surgeries using a state-of-the-art deep learning model. The developed system provides precise segmentation of catheters in medical images, enabling better surgical outcomes.

Key Details

  • Objective: Enhance precision in catheter segmentation for surgical applications.
  • Tools & Technologies: Python, TensorFlow, DSCNet model, Real-life medical datasets.
  • Achievements: Achieved 98% accuracy and an IoU of 0.94.

Workflow

  1. Problem Definition: Identified the challenges of segmenting catheters in complex medical images.
  2. Data Preparation: Preprocessed real-life datasets, including normalization and data augmentation.
  3. Model Development: Implemented the DSCNet deep learning model for high-precision segmentation.
  4. Evaluation: Assessed the model's performance using IoU and accuracy metrics.

Future Work

  • Testing the model on larger and more diverse datasets.
  • Integrating the system into real-time surgical tools.
  • Exploring transfer learning to adapt the model for related medical applications.
